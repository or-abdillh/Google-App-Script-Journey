# Google App Script Journey 🚀

Repositori ini menampung berbagai studi kasus penggunaan **Google App Script (GAS)** untuk mengotomatiskan pekerjaan sehari-hari di ekosistem Google Workspace.

## 📖 Tentang

[Google App Script](https://developers.google.com/apps-script) adalah platform scripting berbasis JavaScript yang memungkinkan kita untuk memperluas dan mengotomatiskan produk Google seperti Sheets, Docs, Drive, Gmail, Calendar, dan lainnya.

## 📂 Daftar Studi Kasus

| No | Judul | Deskripsi |
|----|-------|-----------|
| 01 | [Automated Email Report](./01-automated-email-report/) | Mengirim laporan otomatis via Gmail dari data Google Sheets |
| 02 | [Form Response Handler](./02-form-response-handler/) | Memproses dan mengorganisir respons Google Form ke Sheets |
| 03 | [Google Sheets CRUD](./03-google-sheets-crud/) | Operasi Create, Read, Update, Delete pada Google Sheets |
| 04 | [Google Drive File Manager](./04-google-drive-file-manager/) | Mengelola file dan folder di Google Drive secara otomatis |
| 05 | [Google Calendar Automation](./05-google-calendar-automation/) | Membuat dan mengelola event Google Calendar secara otomatis |

## 🛠️ Cara Menggunakan

1. Buka [Google Apps Script](https://script.google.com/) dan buat project baru.
2. Pilih studi kasus yang ingin dicoba.
3. Salin isi file `Code.gs` dari folder studi kasus ke editor Apps Script.
4. Sesuaikan konfigurasi (ID spreadsheet, email, dsb.) sesuai kebutuhan.
5. Jalankan fungsi yang diinginkan melalui menu **Run** atau trigger otomatis.

## 📋 Prasyarat

- Akun Google aktif.
- Akses ke [Google Apps Script](https://script.google.com/).
- Pemahaman dasar JavaScript.
